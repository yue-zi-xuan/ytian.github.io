    Maven的使用

    Maven的基本命令

    compile：编译

    将java源文件编译成.class文件

    test ：测试项目

    运行test目录下的测试用例

    package ：项目打包

    打包为xxx, xxx在xxx中指定

    clean ：删除target目录

    install ：安装

    将当前项目放入maven本地仓库中，供其它项目使用

    关于“坐标”

    groupId和artifactId构成了一个Jar包的坐标

    • groupId：所需Jar包的项目名
    • artifactId：所需Jar包的模块名

    依赖范围scop

    1. compile 默认scop，表示当前依赖需要参与项目的编译；打包时会包含进去
    2. provided 只会在编译，测试阶段起作用；依赖不会打入jar包中
    3. runtime 参与运行阶段，打包时会包含
    4. test 只在test域中起作用；打包时不会包含
    5. system 表示使用系统目录下的依赖，需要和systemPath一起使用
    6. optional 可选的
